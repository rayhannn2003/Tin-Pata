# Known Limitations

Tin Pata **v2.0.0-alpha.4** — personal-use, offline-first Android app.

---

## Platform & build

| Limitation | Detail |
|------------|--------|
| **Expo Go unsupported** | PDF reader and brightness require a **development or production build** (`npx expo run:android` or EAS APK). |
| **Android-first / local-first** | Primary target is Android. Core reading works offline without an account. iOS may build but is not release-tested. |
| **Native folders not committed** | `android/` and `ios/` are generated by `expo prebuild` locally. See [CNG_SETUP.md](./CNG_SETUP.md). |
| **Optional cloud** | Auth + metadata sync + PDF cloud backup require Supabase setup. No forced login. |

---

## PDF reader

| Limitation | Detail |
|------------|--------|
| **Black flash on resume** | Opening at a saved page may briefly flash black (`react-native-pdf` native behavior). |
| **Fit/scroll apply at open only** | Fit and scroll preferences apply once when the reader opens. |
| **Horizontal scroll varies by PDF** | Page-by-page mode may behave differently per PDF. |
| **Focus mode is UI-only** | Hides app controls; does not modify the PDF renderer. |
| **No PDF text search** | Not planned for v1.4. |
| **Large PDFs** | Very large files may load slowly. |

Details: [READER_KNOWN_LIMITATIONS.md](./READER_KNOWN_LIMITATIONS.md)

---

## Data & backup

| Limitation | Detail |
|------------|--------|
| **JSON backup does not include PDF files** | Export includes reading metadata, sessions, notes, bookmarks, goals, reflections, and portable settings — **not PDF bytes**. |
| **Restored books may need PDF relink** | After import on a new device, use **Relink PDF** or **Download PDF** (if cloud backup exists). Progress, notes, and bookmarks are preserved. |
| **Optional PDF cloud backup** | Manual per-book upload to Supabase Storage (max **50 MB**). Not included in JSON backup. See [V2_PDF_CLOUD_BACKUP.md](./V2_PDF_CLOUD_BACKUP.md). |
| **Metadata sync ≠ PDF sync** | Sync now pushes reading metadata only. PDF upload/download is separate. |
| **Old backups compatible** | Missing fields use safe defaults on import; cloud PDF flags reset to local-only. |

---

## Reader comfort

| Limitation | Detail |
|------------|--------|
| **Brightness needs native module** | `expo-brightness` must be in the built APK. |
| **Brightness is system-level** | Affects device/app brightness, not PDF rendering internals. |
| **Keep awake is reader-only** | Screen stays on while the reader screen is open when enabled. |

---

## Stats & insights (v1.2+)

| Limitation | Detail |
|------------|--------|
| **Insights need session data** | Mood, focus, and blocker insights need finish-session survey data. |
| **Estimated finish is approximate** | Based on average pages per reading day. |
| **No chart library** | Insights are card-based summaries. |
